# Harmonie et Mieux-Être

Site vitrine de **Céline Belin**, praticienne en soins énergétiques (humains et animaux).

- **Domaine :** https://www.harmonie-et-mieux-etre.be
- **Repo :** https://github.com/rdgdeg/celine-mieuxetre

## Stack

React 19 · Vite · TypeScript · Tailwind CSS · React Router · GSAP

## Développement

```bash
npm install
npm run dev
```

## Build production

```bash
npm run build
```

Génère `public/sitemap.xml` puis compile TypeScript et bundle Vite.

## Contenu

- 9 pages méthodes (`/methodes/:slug`)
- Blog (6 articles)
- Pages informatives : quand consulter, première séance, éthique, à propos, FAQ, contact
- SEO : meta tags dynamiques, JSON-LD, sitemap
